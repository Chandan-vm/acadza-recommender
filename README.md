# Acadza AI Intern Assignment

## Objective
Build a recommender system that analyzes student performance and suggests what to study next.

## Key Highlight
This system is designed to be extensible and can be upgraded to ML-based recommendations.

## Approach
- Identify weak topics (score < 50)
- Generate step-by-step learning path
- Provide targeted questions per level

## Tech Stack
- Python
- CSV data handling

## Project Structure
- data/ → dataset
- src/ → core logic
- debug/ → debugging explanation
